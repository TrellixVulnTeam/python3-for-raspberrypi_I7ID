# python3-for-raspberrypi
python3 for raspberry pi.


Python3.6.3

python3.6.3 build on raspberry pi 
(2017-09-07-raspbian-stretch-lite)

usage:
git clone this-raspo
cd Python-3.6.3
sudo make install
sudo mkdir /usr/include/python3.6
sudo cp -rf Include/* /usr/python3.6
